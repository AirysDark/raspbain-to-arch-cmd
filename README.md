# apt-arch-wrapper

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/AirysDark/apt-arch-wrapper/main/install.sh)
```

## Uninstall
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/AirysDark/apt-arch-wrapper/main/uninstall.sh)
```

## Usage
```bash
apt help
apt update
apt install htop -y
apt echo on
apt echo off
```
